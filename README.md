# 🍽️ Restaurant API (.NET)

A simple **RESTful Web API** built using **ASP.NET Core**, **Entity Framework Core**, and **SQL Server**.

This project demonstrates:

- JWT Authentication
- Clean project structure
- Entity Framework Core integration
- Secure REST API development

This project is intended for **learning backend development concepts and API architecture**.

---

# 🚀 Features

- User Registration
- User Login with JWT Authentication
- Secure API endpoints
- SQL Server database integration
- Entity Framework Core ORM
- RESTful API architecture
- Swagger API documentation

---

# 🛠️ Tech Stack

**Backend**

- ASP.NET Core Web API
- C#

**Database**

- SQL Server
- Entity Framework Core

**Authentication**

- JWT (JSON Web Token)

**API Testing**

- Postman
- Swagger
